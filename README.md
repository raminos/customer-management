# Customer Management

## Specification:
- **Overview page:**
    - Lists all customers
    - Add a new customer
    - Link to edit a customer (‘Customer Detail’ page)
    - Delete a customer
- **Customer Detail page:**
    - Edit & remove profile data

## Bonus:
- Setup a node server
- Use persistent data storage (MongoDB, ...)
- Host the application (Heroku, ...)
- Continuous Integration & Development (Jenkins & Travis)
